Новые открытия в космосе

Учёные обнаружили новую экзопланету в зоне обитаемости звезды Kepler-452. Планета Kepler-452b имеет радиус в 1.6 раза больше Земли и обращается вокруг своей звезды за 385 дней. Условия на планете могут быть подходящими для существования жидкой воды, что делает её потенциальным кандидатом для поиска внеземной жизни.
