# Лабораторная работа: PWA — Менеджер паролей

Прогрессивное веб-приложение (PWA), которое можно установить как мобильное приложение.

## Функционал
- Локальное хранение паролей в localStorage браузера
- Генерация сложных паролей (16 символов: буквы, цифры, спецсимволы)
- Добавление и удаление записей
- Работает оффлайн благодаря Service Worker
- Устанавливается на рабочий стол/телефон

## Как запустить
1. Открыть `index.html` через локальный сервер (например, Live Server в VS Code или `python -m http.server`)
2. В Chrome нажать на кнопку установки в адресной строке
3. Приложение откроется в отдельном окне и будет работать оффлайн

Технологии: HTML, CSS, JavaScript (vanilla), PWA (manifest.json + service-worker.js)
